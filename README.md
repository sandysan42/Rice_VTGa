# Rice Voxel Grid Tracing algorithm

Welcome to the public repository for the code used in our paper for paper A novel high-throughput digital morphological phenotyping method for evaluating growth traits in rice, This repository contains the code that was used to conduct the experiments and produce the results presented in the paper (3D point cloud model of rice (in stanford PLY format))
Link to paper: https://doi.org/10.1002/ppj2.70054
